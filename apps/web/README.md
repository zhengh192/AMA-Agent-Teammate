# AMA Web

Phase 3 React/Vite UI for Phase 1/2 chat and analysis plus uploads, ingestion status, source citations, Knowledge conflicts, Skill/Memory proposal diffs, exact approval, deprecation, rollback, expiry, and deletion.

```powershell
pnpm --dir apps/web dev
```

The development server proxies `/api` to `http://127.0.0.1:8000`.
